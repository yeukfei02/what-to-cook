# what-to-cook

what-to-cook

## Requirement

- install python (v3.12)

```zsh
// create virtualenv
$ python -m venv .venv

// activate virtualenv
$ source .venv/bin/activate
or
$ sh .venv/bin/activate

// install dependencies
$ pip install -r requirements.txt

// run in local
$ python app.py
```

```zsh
// copy .env file
$ cp .env.sample .env
```
