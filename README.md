# what-to-cook

what to cook is a multi llm agent for user to find what to eat or what to cook by searching different ingredients.

## Tech Stack

- python
- uv
- strands agent
- bedrock agentcore
- nicegui

ai agents: `packages/ai_agents`

web: `packages/web`

## Requirement

- install python (v3.12)

```zsh
// install dependencies
$ uv sync

// run in local
$ uv run app.py
```

```zsh
// copy .env file
$ cp .env.sample .env
```
